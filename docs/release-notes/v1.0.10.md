## What's New

- Dashboard onboarding now reflects the actual setup order: install the memoQ DLL first, configure the provider next, then build profiles, and finally review translation history.
- History Explorer now supports deleting translation records from the desktop UI, including single-entry deletion and bulk deletion for selected rows.
- README content in English and Simplified Chinese now matches the shipped product surface instead of implying currently unexposed modules.

## UI

- Long values that are likely to be clipped now expose the full text on hover in key areas such as history tables, history details, dashboard descriptions, side-panel profile names, and provider URLs.
- History details and table actions are more maintainable because deletion is now wired end to end through the runtime persistence layer.

## Notes

- This release continues to use the existing Windows packaging and update flow.
- Release packaging still requires `pnpm`, `node`, and `.NET SDK` to be available to the packaging shell environment.
