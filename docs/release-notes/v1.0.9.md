## What's New

- Portable packages stay smaller with `app.asar`, runtime-only dependency copying, and Electron locale pruning for English and Simplified Chinese.
- Portable update UX now opens the browser release page instead of offering in-app install-style update actions.
- This release fixes the withdrawn `v1.0.8` packaging regression by preserving transitive runtime dependencies required by the desktop background worker.

## Packaging

- The portable package keeps only the runtime dependencies the app actually needs, including transitive dependencies required at launch time.
- Portable zip size and unpack time remain lower than `v1.0.7` while restoring the integration install flow in packaged builds.

## Notes

- This release is still distributed as a portable Windows package.
- Portable updates direct users to the GitHub release download page for a clearer and safer upgrade path.
