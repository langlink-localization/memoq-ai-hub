## What's New

- Portable packages are smaller because the build now keeps only the runtime dependencies the app actually loads and prunes Electron locales down to English and Simplified Chinese.
- Portable update UX now opens the browser release page instead of offering in-app install-style actions.
- The app explicitly explains that the browser download flow is the intended portable upgrade path for a clearer and safer user experience.

## Packaging

- Portable package contents now use `app.asar`, which cuts small-file overhead and reduces unpack time.
- The packaged portable zip is smaller than `v1.0.7`, and the unpacked app footprint is reduced as well.

## Notes

- This release is still distributed as a portable Windows package.
- Portable updates now direct users to the GitHub release download page rather than trying to update inside the app.
