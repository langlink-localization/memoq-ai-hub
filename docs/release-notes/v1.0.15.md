## What's New

- Added a dedicated Logs page for support and development diagnostics, including log directory details, retention policy, grouped log files, refresh, folder actions, cleanup, and diagnostic summary copy.
- Added rotating runtime logs across the desktop app, worker, gateway, runtime, renderer error bridge, preview helper, and memoQ plugin with 5 MB file rotation, 6-file source retention, 14-day cleanup, and sensitive-data redaction.
- Refined desktop UI copy and layout in English and Chinese so ordinary labels and descriptions are easier to read, while long paths and IDs remain available without crowding the interface.

## Notes

- Logs are intended for troubleshooting and avoid recording source text, target text, prompt bodies, API keys, and full provider payloads.
- The Chinese locale is now maintained as a full resource file instead of reusing English strings.
- This release updates the memoQ plugin logging behavior; reinstall the memoQ integration DLL after upgrading if it was already installed.
