## What's New

- Update checks now use a 12-second manifest timeout, so the Dashboard no longer stays on Checking when the network is slow or unreachable.
- The Updates card now keeps the latest known version readable while checking and records same-version manifests as up to date.
- Update-check errors now show friendly English and Chinese messages while structured diagnostics go to the local logs.
- Select controls in the main content, forms, drawers, and cards now fill their container more consistently.
- Select dropdown panels keep long provider, model, asset, and path names readable with wrapping and a 90vw viewport cap.

## Notes

- Package downloads keep the existing behavior; the short timeout applies to the small update manifest request.
- This release includes the v1.0.15 logging, localization, and UI-copy work plus the follow-up update-center and Select width fixes.
