## What's New

- Restored recursive runtime dependency packaging for the portable desktop app so the background worker keeps all required transitive modules at launch time.
- Fixed packaged desktop version resolution so shipped builds report `1.0.12` instead of falling back to an unrelated historical default.
- Added release workflow protection that rejects tags whose commits are not reachable from `main`, preventing stale release branches from publishing new public versions.

## Notes

- `v1.0.11` remains published for audit purposes but is withdrawn and should not be used.
- Stable update metadata should point new downloads at `v1.0.12`.
