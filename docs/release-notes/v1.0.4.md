# memoQ AI Hub v1.0.4

## Highlights

- Added in-app update checks for new desktop releases.
- Added a portable-friendly upgrade flow that downloads and prepares a new version without replacing the running app.
- Added installer release assets and update metadata for future installed-app upgrades.

- Update service, runtime, background worker, repository layout, and release metadata checks passed locally.

## Upgrade Notes

- Portable builds now guide users to switch to a prepared new version directory instead of overwriting the current executable.
- If a release updates memoQ plugin files, reinstall the integration from the desktop app after upgrading.
