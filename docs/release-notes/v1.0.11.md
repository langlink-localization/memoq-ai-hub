## Status

- Do not use this release.
- The published `v1.0.11` assets were built from an incorrect release line and can surface two high-impact regressions:
  - The portable app may report its version as `v1.0.8` after unzip.
  - Provider draft tests can fail with `Desktop background worker exited with code 1`.

## Replacement

- Use `v1.0.12` or newer instead.
- `v1.0.12` restores the portable runtime dependency packaging required by the background worker and repairs packaged desktop version resolution.

## Notes

- Keep this withdrawn release published so existing download links remain traceable, but do not promote or reuse its assets.
