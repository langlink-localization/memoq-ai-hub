## What's New

- Compatible providers now start without a guessed default model and guide users to discover or add the correct model first.
- Provider test status is clearer and stays consistent across the page before saving.
- Builder now preserves the `Use cache` setting correctly and adds a sticky save bar for long pages.

## Packaging Fix

- Fixed the release packaging path after the broken `v1.0.6` run.
- Upgraded GitHub Actions `pnpm/action-setup` from `v4` to `v5` to move off the deprecated Node 20 action runtime.

## Notes

- This release is still distributed as a portable Windows package.
- `v1.0.6` should be treated as invalid because it published source code only.
